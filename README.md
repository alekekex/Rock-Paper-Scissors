# Rock Paper Scissors

A console-based Rock Paper Scissors game written in Java. This project was created to practice object-oriented programming and design.

## Features

- Single-player gameplay against the computer
- Best-of-1, Best-of-3, and Best-of-5 game modes
- Random computer choice
- Input validation
- Session statistics tracking
- Replay option

## Technologies

- Java
- IntelliJ IDEA
- Git

## How to Run

### Option 1: IntelliJ IDEA

1. Clone or download the repository.
2. Open the project in IntelliJ IDEA.
3. Run `Main.java`.

### Option 2: Command Line

1. Clone the repository:

```bash
git clone https://github.com/alekekex/Rock-Paper-Scissors.git
```

2. Change to the project directory:

```bash
cd RockPaperScissors
```

3. Compile the program:

```bash
javac Main.java
```

4. Run the program:

```bash
java Main
```

Make sure Java is installed before running the program.
